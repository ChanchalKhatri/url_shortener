# 🔗 URL Shortener

A modern URL Shortener built with **React (Vite)** that converts long URLs into short links using the TinyURL API.

## 🚀 Features

* Shorten long URLs
* Copy shortened links
* Delete URL history
* LocalStorage support
* Responsive UI
* Smooth animations

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* Framer Motion
* Lucide React
* Axios
* TinyURL API

## 📦 Installation

```bash
npm install
npm run dev
```

## 🔗 API Used

```text
https://tinyurl.com/api-create.php
```

## 💾 LocalStorage

URL history is stored locally in the browser and remains available after page refresh.
